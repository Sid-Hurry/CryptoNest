# Cryptonest

Cryptonest is a modern cryptocurrency tracking web application designed to help users monitor live market data, analyze individual coins, and compare cryptocurrencies through a clean and intuitive interface. The application supports light and dark themes and is fully responsive across devices.

---

## Overview

Cryptonest provides real-time cryptocurrency insights using reliable public APIs. Users can explore market trends, view detailed coin statistics, compare assets side by side, and track overall market performance without requiring authentication.

---

## Key Features

- Real-time cryptocurrency price tracking  
- Global market overview including total market cap and dominance  
- Side-by-side coin comparison with detailed metrics  
- Individual coin pages with interactive price charts  
- Smart search with live suggestions  
- Light and dark theme toggle  
- Fully responsive design  

---

## Technology Stack

- React.js  
- React Router DOM  
- Tailwind CSS with CSS variables  
- React Context API  
- react-google-charts  
- CoinGecko API  
- Vercel  

---

## Getting Started

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/Sid-Hurry/CryptoNest.git
cd CryptoNest
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

---

## API Usage

Cryptonest uses the public CoinGecko API to fetch cryptocurrency market data including prices, market capitalization, volume, and historical trends. No authentication is required.

---

## Deployment

The application is deployed using Vercel. You can deploy your own version by importing the repository into Vercel and using the default React build configuration.
